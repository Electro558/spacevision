🔧 Feature 1: CSG Boolean Operations

TinkerCAD Mode (Default)

  • Multi-select objects (Shift+Click)
  • Mark shapes as "Solid" or "Hole"
  • Click "Group" → solids union, holes subtract
  • "Ungroup" to revert (non-destructive)
  • Grouped objects become a single mesh

Advanced Mode (Toggle)

  • Select exactly 2 objects
  • Choose: Union / Subtract / Intersect
  • Preview result before confirming
  • Result replaces both source objects
  • Full undo/redo support
Data Model Change: SceneObject gets new fields:
isHole: boolean — marks shape as subtractive
groupId: string | null — links objects in a CSG group
csgResult: BufferGeometry | null — cached boolean result
groupChildren: string[] | null — for compound objects

🎯 Feature 2: Multi-Select System

Required foundation for both CSG and Alignment features

  • Shift+Click — Add/remove from selection
  • Ctrl+A / Cmd+A — Select all
  • Drag box select — Marquee selection on viewport
  • selectedId: string | null → selectedIds: Set<string>
  • All existing single-select features continue working (properties panel shows first selected)
  • Multi-select toolbar appears when 2+ objects selected

🧲 Feature 3: Smart Snapping System

Grid Snapping (Enhanced)

  • Configurable increments: 0.1, 0.25, 0.5, 1.0
  • Visual grid updates to match snap size
  • Dropdown selector in toolbar
  • Separate rotation snap (15°, 30°, 45°, 90°)

Snap-to-Face / Workplane

  • Raycast to detect hovered face
  • Highlight target face in blue
  • New object spawns on face center
  • Oriented to face normal

Snap-to-Edge / Vertex

  • Detects nearest edge/vertex while dragging
  • Green highlight on snap point
  • Configurable snap distance threshold
  • Works with all primitive types

Visual Snap Guides

  • Dashed lines showing alignment with other objects
  • Center-to-center alignment indicators
  • Edge-to-edge alignment indicators
  • Auto-fade when not near snap threshold

📐 Feature 4: Align & Distribute Tools

Alignment (select 2+ objects)

  • Align Left / Center / Right (X axis)
  • Align Front / Center / Back (Z axis)
  • Align Top / Center / Bottom (Y axis)
  • Keyboard: Alt+L/C/R for X, Alt+T/M/B for Y

Distribution (select 3+ objects)

  • Distribute evenly on X / Y / Z axis
  • Equal spacing between bounding boxes
  • Keyboard: Alt+Shift+X/Y/Z

📏 Feature 5: Smart Rulers & Dimensions

  • Object dimensions overlay — Shows W × H × D on selected object
  • Distance measurement — Select 2 objects → shows distance between them
  • Dimension input — Click a ruler label to type exact value
  • While dragging — Shows distance from origin and distance to nearest objects
  • Toggle — Keyboard shortcut M to show/hide measurements
